Stage 5　ページをアップロードする

前回まででページが出来上がりました。せっかくページを作ったので公開していきましょう。今回使うのは GitHub Pages というサービスです。メールアドレスなどでアカウントを作成すれば、最初から最後まで全体を通して無料でできます。とはいえ、この方法は巷にいくらでも情報が転がっていますので、そちらをご覧ください。

え？「教える詐欺」ですって？じゃあネットの検索で「どうやったらその情報にたどり着けるか」お教えしますよ。

まずは

GitHub 入門

で検索して、GitHub が一体何者で、どういう仕組みかをおさえます。ある程度理解したら、今度は

GitHub ウェブ 初心者

で、GitHub Pages というものの存在を含め、どうやれば GitHub を使ってウェブページをアップロードできるか、頭に入れてください。その際、「GitHub 入門」で検索したことが役に立つと思います。

こうして GitHub というものの理屈がわかったら、あとは皆さんの手でページをアップロードするだけです。がんばれ。

最後まで終えた方へ

これで皆さんのウェブページは完成です。お疲れ様でした。

今回 GitHub に登録した方は、まったく同じシステムでプログラミングコードの保存や共有なんかができてしまいます。GitHub の使い道として一番大きいのが、複数人で同じコードを弄ることでしょう。ぜひウェブページに限らず、様々なプログラミング言語に対応しています。しかも、コードだけでなく仕様書なども共有できるので、まあ便利なんですわ。ぜひ使いこなしてくださいね。
