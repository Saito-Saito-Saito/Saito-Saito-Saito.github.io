各種メソッド

random()

0 から 1 の間から float 型をランダムにリターンする。

                    
import random
for i in range(5):
    print(random.random())
                    
                
                    
0.7935651373830874
0.20218037021719304
0.5168870571724608
0.4512803860271539
0.0012269679018672708

choice()

list 型の引数の中にある要素からひとつランダムに選択してリターンする。

                    
import random
for i in range(5):
    print(random.choice([1, 2, 3, 4, 5]))
                    
                
                    
2
1
1
2
5
