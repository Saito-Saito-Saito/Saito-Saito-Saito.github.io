レベル

タスク最適なレベル
問題を診断するときのみに関心のある詳細な情報DEBUG
想定通りのことが起こったことの確認INFO
想定外のことが起こった、もしくは起こりそうなことの警告WARNING
より重大な問題によって、ソフトウェアがある機能を実行できないことの表示ERROR
プログラム自体が実行できないような重大なエラーCRITICAL

デフォルトのレベルは WARNING です。つまり特段設定をしなければ WARNING, ERROR, CRITICAL のレベルのログが出力されます。

また、それぞれのレベルのログを出力するメソッドもあります。

レベルメソッド
DEBUGlogging.debug()
INFOlogging.info()
WARNINGlogging.warning()
ERRORlogging.error()
CRITICALlogging.critical()

以下のコードを参考にして使ってください。

logging.warning('There is not enough memory')

basicConfig

基本的にはこんなスタイルになります。

logging.basicConfig(level=logging.DEBUG, format='%(asctime)s %(levelname)s - %(filename)s - %(lineno)d - %(message)s')

basicConfig は debug() や info() といった呼び出しよりも

前に

呼び出す必要があります。
