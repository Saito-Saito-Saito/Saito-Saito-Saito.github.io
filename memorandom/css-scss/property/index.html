... プロパティのメモ

height

要素の高さを指定する。

width

要素の幅を指定する。

color

文字色を指定する。

background-color

背景色を指定する。

margin

マージン（ボーダーの外側に広がる余白）を設定する。

padding

パディング（ボーダーの内側に広がる余白）を設定する。

font-size

フォントの大きさを指定する。

line-height

文章の 1 行の高さを指定する。

cursor

カーソルが要素の上に乗った時の形状を指定する。

text-align

文章の位置どりを指定する。

position

要素の位置どりを指定する。

visibility

要素の位置を確保したままで、要素を隠すかどうかを指定する。

display

要素の見せ方を指定する。

top

display: absolute とした要素と当該の要素の上辺をどれだけ離すか指定する。

right

display: absolute とした要素と当該の要素の右辺をどれだけ離すか指定する。

float

要素のフロート（横並び）を指定する。

z-index

要素の重複の優先順位を指定する。

list-style

リストの要素の前に付される目印の形状を指定する。

transition

要素の動きを指定する。

transition-delay

transition プロパティで指定した挙動の開始時間の遅延時間を指定する。
