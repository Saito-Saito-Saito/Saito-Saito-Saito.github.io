擬似クラスとは

セレクタが特定の状態であるときだけに指定されるスタイルシートである。例えば id が example の要素の上にマウスが乗ったとき、スタイルシートに

                    
#example:hover
{
    opacity: 30%;
}

とすれば、この要素は普段しっかり見えていても、マウスを乗せると 30% まで落ちてしまう。

一覧

:not()

() の中に入っているセレクタ意外に適用するプロパティを指定する。

:hover

カーソルが要素の上に乗ったときの挙動を指定する。

:active

要素がクリックやタップされたときの挙動を指定する。
